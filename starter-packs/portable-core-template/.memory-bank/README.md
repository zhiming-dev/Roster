# Memory Bank

This folder stores local context that helps the coding agent maintain continuity across sessions.

## Files

- `activeContext.md`: current focus, progress, and next steps
- `learnings.md`: durable technical discoveries and architecture notes
- `userDirectives.md`: optional user preferences and collaboration style

## Workflow

- Read `activeContext.md` and `learnings.md` at session start.
- Update files after meaningful progress or decisions.
- Keep entries concise and specific to this repository.

## Privacy

These files are intended to be local to each developer unless your team explicitly chooses to share them.
